# TPT Virtual Assistant Agent

An automated agent for uploading educational resources to Teachers Pay Teachers (TPT).

## Project Goals

- Automate the document upload process to TPT
- Handle metadata entry (titles, descriptions, pricing, subjects, grade levels)
- Reduce manual repetitive work for TPT sellers
